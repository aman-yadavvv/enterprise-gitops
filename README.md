# SoleStyle - Modern Sneaker Landing Page

A full-stack e-commerce landing page for sneakers built with the MERN stack (MongoDB, Express.js, React, Node.js) with modern animations and responsive design.

## 🚀 Features

### Frontend
- ⚡ **React 18** with Vite for blazing fast development
- 🎨 **Tailwind CSS** for modern, responsive styling
- ✨ **Framer Motion** for smooth animations
- 🔄 **React Router** for seamless navigation
- 📦 **React Query** for efficient data fetching
- 🛒 **Cart Management** with context API
- 🔐 **Authentication** with JWT
- 📱 **Fully Responsive** design
- 🎯 **SEO Friendly** with meta tags
- 🌙 **Modern UI/UX** with glass-morphism effects

### Backend
- 🚀 **Node.js** with Express.js
- 🗄️ **MongoDB** with Mongoose ODM
- 🔑 **JWT Authentication** & Authorization
- 🛡️ **Helmet** for security headers
- ⚡ **Rate Limiting** for API protection
- 📧 **Nodemailer** for email services
- 💳 **Stripe** payment integration
- 🖼️ **Cloudinary** for image uploads
- 🐳 **Docker** support for containerization

### Database Models
- Users with role-based access
- Products with categories, sizes, colors
- Orders with status tracking
- Cart with real-time updates
- Reviews and ratings
